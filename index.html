GET/api/downloader/tiktok
Download TikTok videos without watermark. Returns direct CDN URL.
Params: url (required) · quality (hd/sd)
// Example Request fetch('https://api.aetheria.my.id/downloader/tiktok?url=VIDEO_URL') .then(r => r.json()) .then(data => console.log(data));
POST/api/ai/chat
AI Chat completion powered by GPT-4 backend. Streaming supported.
Body: message · system · stream
fetch('https://api.aetheria.my.id/ai/chat', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ message: 'Hello!' }) }).then(r => r.json()).then(console.log);
GET/api/downloader/youtube
YouTube video & audio downloader. Supports mp4/mp3 with quality selection.
Params: url · format · quality
fetch('https://api.aetheria.my.id/downloader/youtube?url=YT_URL&format=mp4') .then(r => r.json()).then(data => console.log(data));
GET/api/tools/qrcode
Generate QR Code from any text or URL. Returns image buffer.
Params: text · size · color
fetch('https://api.aetheria.my.id/tools/qrcode?text=Hello+World') .then(r => r.blob()) .then(blob => { /* render image */ });
GET/api/search/google
Google search scraper returning structured JSON with snippets & links.
Params: q · limit · lang
fetch('https://api.aetheria.my.id/search/google?q=aetheria&limit=5') .then(r => r.json()).then(data => console.log(data));
GET/api/anime/info
Anime info, episodes & streaming links from MyAnimeList.
Params: title · mal_id
fetch('https://api.aetheria.my.id/anime/info?title=Naruto') .then(r => r.json()).then(data => console.log(data));
GET/api/tools/sticker
Convert image/gif to WhatsApp sticker with custom metadata.
Params: url · pack · author
fetch('https://api.aetheria.my.id/tools/sticker?url=IMG_URL&pack=Aetheria') .then(r => r.json()).then(data => console.log(data));
POST/api/tools/imgai
AI image generation from text prompt. Returns CDN URL.
Body: prompt · style · size
fetch('https://api.aetheria.my.id/tools/imgai', { method: 'POST', body: JSON.stringify({ prompt: 'futuristic city neon' }) }).then(r => r.json()).then(console.log);
