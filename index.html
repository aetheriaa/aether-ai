📖 Panduan Setup API

1. Pterodactyl — Application API Key

Login panel sebagai admin → Settings → API → Application API → Create New

Format: ptla_xxxxxxxxxxxxxxxxx


2. Node ID & Allocation ID

Buka Nodes → klik node → lihat ID di URL.
Untuk Allocation ID: klik tab Allocation di halaman node, lihat ID kolom pertama.


3. Egg ID

Buka Nests → klik nest → klik egg → lihat ID di URL atau tab Configuration.



4. Pakasir

Daftar di app.pakasir.com → buat project → salin slug & API key.

Pastikan metode pembayaran yang ingin kamu pakai sudah diaktifkan di dashboard Pakasir.



⚠ API key Pterodactyl memberikan akses penuh ke panel. Jangan bagikan ke siapapun.


5. Alur Otomatis

Customer bayar → Pakasir kirim notifikasi / polling terdeteksi lunas → sistem buat user Pterodactyl → buat server → tampilkan kredensial ke customer.

Jika API Pterodactyl gagal (CORS/network), order tetap tercatat sebagai active di admin dan kamu bisa buat server manual dari dashboard Pterodactyl.
